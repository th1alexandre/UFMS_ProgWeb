## Executar localmente
### Requisitos
#### Windows ou Mac
- Docker Desktop
#### Linux
- Docker
- Docker Compose

<br>

### Variáveis de ambiente

- Copie o conteúdo do arquivo `.env.dev` para um novo arquivo `.env` e atualize as variáveis.

<br>

### Construção da imagem

```
docker-compose build
```

### Executar

```
docker-compose up
```

### Terminar execução

```
docker-compose down
```
